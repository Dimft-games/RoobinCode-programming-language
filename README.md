RoobinCode is small open-source programming  language written on Python.
FUNCTIONS
1. write [text]
2. var [Variable name] - create var
3. set [Variable name]=[text]
4. min [First num]-[Second num] - minus
5. plus [First num]+[Second num]
6. mul [First num]*[Second num] - multiplication
7. div [First num]/[Second num] - division
8. wait [seconds]
